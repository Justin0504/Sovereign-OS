# Charter templates

Drop-in YAML templates for different “digital corp” personas. Use with:

```bash
sovereign run --charter charters/The_Freelancer.yaml "your goal"
```

| Charter | Mission |
|---------|---------|
| **The_Freelancer.yaml** | Open-source bounty hunter: find GitHub issues, implement, submit PRs. Budget-focused. |
| **The_Influencer.yaml** | Social growth: content calendar, posting, engagement metrics. Ad/API budget limits. |
| **The_Analyst.yaml** | Financial risk: ingest data, run risk metrics, produce audit reports. Regulatory limits. |

See [docs/CHARTER.md](../docs/CHARTER.md) for schema (mission, core_competencies, fiscal_boundaries, success_kpis).
