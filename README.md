# Sovereign-OS

**One command. One Charter. A digital corporation that thinks, spends, and answers for every token.**

Not another chatbot. Not another вҖңAI agent framework.вҖқ Sovereign-OS is the **constitution-first substrate**: one YAML defines who the entity is, what it may spend, and how success is measured. The CEO plans. The CFO gates. The Auditor judges. **The Ledger never lies.**

## вҡЎ Quick Start

**CLI вҖ” 30 seconds:**

```bash
git clone https://github.com/Justin0504/Sovereign-OS.git && cd Sovereign-OS
pip install -e .
sovereign run --charter charter.example.yaml "Summarize the market in one paragraph."
```

You get: task plan вҶ’ CFO approval вҶ’ execution вҶ’ audit report. Done.

**Web Dashboard вҖ” paid jobs, 24/7:**

```bash
pip install -e ".[llm]"
# Set STRIPE_API_KEY + OPENAI_API_KEY or ANTHROPIC_API_KEY in .env
python -m sovereign_os.web.app
# Open http://localhost:8000 вҖ” run missions, approve jobs, watch balance & token usage.
```

**Want to charge for work?** [3-step guide вҶ’](docs/QUICKSTART.md) (Stripe + one LLM key; 13 built-in workers out of the box.)

## рҹҸӣпёҸ Architecture

Five layers. Data flows down; accountability flows back.

```
Charter (who we are) вҶ’ Governance (CEO + CFO) вҶ’ Registry (workers) вҶ’ Ledger (every $ & token) вҶ’ Auditor (pass/fail вҶ’ TrustScore)
```

- **Charter** вҖ” Mission, competencies, fiscal bounds, KPIs. One file.
- **Governance** вҖ” CEO decomposes goals into tasks; CFO approves or denies budget per task.
- **Registry** вҖ” Maps skills to workers (13 built-in: summarize, research, reply, write_article, translate, вҖҰ).
- **Ledger** вҖ” Append-only. Runway, burn rate, income. Never lies.
- **Auditor** вҖ” KPI verification. Pass вҶ’ TrustScore вҶ‘. Fail вҶ’ TrustScore вҶ“.

## рҹҡҖ Features

- **рҹ“Ұ 13 built-in workers** вҖ” summarize, research, reply, write_article, solve_problem, write_email, write_post, meeting_minutes, translate, rewrite_polish, collect_info, extract_structured, spec_writer. No code; configure Stripe + one LLM key and run paid jobs. [QUICKSTART](docs/QUICKSTART.md)
- **рҹ”„ Multi-model** вҖ” Strategist and workers can use different backends (e.g. GPT-4o for planning, cheaper models for execution).
- **рҹ”җ SovereignAuth** вҖ” RBAC by TrustScore. READ_FILES, WRITE_FILES, SPEND_USD, CALL_API gated; agents earn capabilities.
- **рҹҢҗ Web Dashboard (24/7)** вҖ” Run missions, job queue, approve/retry, health, token usage, audit trail. Optional ingest from URL; Stripe charges; webhook on completion.
- **рҹ”Ң MCP native** вҖ” Plug into the same tool graph as the rest of the ecosystem.
- **рҹ“Ҡ Observability** вҖ” OpenTelemetry, Prometheus metrics, verifiable audit trail with `proof_hash`.
